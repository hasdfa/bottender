# Vraksha Bottender

Conversational UI framework (Bottender fork). Monorepo with 7 packages under `@vraksha` scope.

## Quick Reference

| Command         | Description      |
| --------------- | ---------------- |
| `pnpm compile`  | Build TypeScript |
| `pnpm test`     | Lint + tests     |
| `pnpm lint:fix` | Auto-fix lint    |
